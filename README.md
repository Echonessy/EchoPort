# EchoPort

> 接口调试工具 目前支持post。get方式持续迭代开发

## Creat By Echonessy  2018.11.29

## Run Server

``` bash
# install dependencies
npm install

# 运行node
node server.js

访问 http://127.0.0.1:5100

```
